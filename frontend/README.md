# Frontend - Corrugated Box Cost Calculator

This folder contains the frontend application for the Corrugated Box Cost Calculator.

## Setup Instructions

### Prerequisites
- Node.js 14+ and npm (or yarn)
- Backend running on http://localhost:8000

### Installation

```bash
# Install dependencies
npm install

# Start development server
npm start
```

### API Integration

Refer to the main `FRONTEND_INTEGRATION_GUIDE.md` in the root directory for complete API documentation.

### Key Points
- Backend API runs on: `http://localhost:8000`
- Interactive API docs: `http://localhost:8000/docs`
- All endpoints are documented with request/response examples

### Supported Frameworks
- React
- Vue.js
- Angular
- Next.js
- Svelte
- Any frontend framework supporting HTTP requests
